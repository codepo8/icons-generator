    So what's all this then, eh?

    This is just a small tool that allows you to create a zip containing icons for a packaged application (for example for Firefox OS) from any picture.

    All you do is:

    1. Drag and drop an image on the above sqare
    2. Click the "Create Thumbs" button and check out the previews by rolling over them.
    3. Click the "Download Thumbs" button to get your Zip file

    The source code is available on GitHub, updates will be announced there, too.
